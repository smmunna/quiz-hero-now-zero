1. Map

In JavaScript, the map() function is used to create a new array by calling a specific function on each element present in the parent array. Here is an example of how to use the map() function in JavaScript:


const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map((number) => {
return number * 2;
});
console.log(doubledNumbers);

In this example, the map() function is used to create a new array called doubledNumbers which contains the original numbers array with each element doubled. The map() function takes a callback function as an argument, which is called for each element in the array. The callback function returns the new value for each element, which is then added to the new array.
