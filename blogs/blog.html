1. Map

In JavaScript, the map() function is used to create a new array by calling a specific function on each element present in the parent array. Here is an example of how to use the map() function in JavaScript:


const numbers = [1, 2, 3, 4, 5];
const doubledNumbers = numbers.map((number) => {
return number * 2;
});
console.log(doubledNumbers);

In this example, the map() function is used to create a new array called doubledNumbers which contains the original numbers array with each element doubled. The map() function takes a callback function as an argument, which is called for each element in the array. The callback function returns the new value for each element, which is then added to the new array.

2. filter()

In JavaScript, the filter() method is used to create a new array with all elements that pass the test implemented by the provided function.Here is an example of how to use the filter() function in JavaScript:


const numbers = [1, 2, 3, 4, 5];
const filteredNumbers = numbers.filter((number) => {
return number > 2;
});
console.log(filteredNumbers);

In this example, the filter() function is used to create a new array called filteredNumbers which contains only the elements of the original numbers array that are greater than 2. The filter() function takes a callback function as an argument, which is called for each element in the array. The callback function returns a boolean value, which determines whether the element should be included in the new array.

3. Find()


const numbers = [1, 3, 4, 5, 7, 8, 9];
const firstEven = numbers.find((number) => number % 2 === 0);
console.log(firstEven); // Output: 4

In this example, the callback function checks if the current number is even using the modulus operator (%). The find() method returns the first even number it encounters, which is 4. The Array.find() method is useful when you need to find a specific element in an array that satisfies a certain condition.

4. forEach()

The forEach() method is useful when you need to perform an action on each element in an array

const numbers = [1, 2, 3, 4, 5];
numbers.forEach((number) => {
console.log(number);
});

In this example, the callback function logs the current element to the console on each iteration of the loop.

5. QuizTimer()

This function sets up a countdown timer on our quiz. It takes a single argument, 'dismiss', which is a boolean that determines whether the timer should be stopped or not. It takes the value 'true' or 'false' only. If dismiss is true, the function clears the interval using clearInterval(timer) and returns, effectively stopping the timer. else If dismiss is false or empty, the function starts the timer using setInterval(). The timer function increments a count variable every second, updates the HTML of a DOM element with id count, and changes the color of the text every 60 seconds.

6. chooseQuiz()

The chooseQuiz function takes two parameters: 'index' and 'givenAns'. It first checks if an answer with the same id as the current question exists in the answers array. If it exist, it updates the existing answer with the new givenAns. If it doesn't exist, it adds a new object with the id, question, and givenAns to the answers array. Finally, it calls the displayAnswers function to display the updated answers array.
